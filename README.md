# Installation
- Requires latest Reframework [REFramework Nighly Release](https://github.com/praydog/REFramework-nightly/releases)
- Drag BowChargeMultiplier.lua inside SteamLibrary\steamapps\common\MonsterHunterWilds\reframework\autorun

# Technical Details
- Hooked Method: app.cHunterWp11Handling.updateCharge
- Field Modified: _ChargeTimer

# Notes

- Recommended multipliers: x1.35 – x3.
- Setting excessively high multipliers (>1.35) may cause unexpected behavior / instant Charge.
